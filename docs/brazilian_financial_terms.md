# Glossário de Termos Financeiros e de Investimento para o Brasil (pt-BR)

Este documento visa consolidar termos financeiros e de investimento específicos do mercado brasileiro, relevantes para a plataforma.

| Termo (Português)                 | Sigla/Abreviação | Inglês (Referência/Equivalente Comum) | Notas Breves                                                                 |
| --------------------------------- | ---------------- | ------------------------------------- | ---------------------------------------------------------------------------- |
| Tesouro Direto                    | TD               | Direct Treasury (Brazilian Gov. Bonds) | Plataforma de negociação de títulos públicos federais para pessoas físicas.   |
| Certificado de Depósito Bancário  | CDB              | Certificate of Deposit (Bank)         | Título de renda fixa emitido por bancos.                                       |
| Letra de Crédito Imobiliário      | LCI              | Real Estate Credit Bill               | Título de renda fixa lastreado em créditos imobiliários, isento de IR para PF. |
| Letra de Crédito do Agronegócio   | LCA              | Agribusiness Credit Bill              | Título de renda fixa lastreado em créditos do agronegócio, isento de IR para PF.|
| Fundo de Investimento Imobiliário | FII              | Real Estate Investment Trust (REIT)   | Fundo que investe em ativos imobiliários. Cotas negociadas em bolsa.          |
| Ibovespa                          | IBOV             | Ibovespa (Brazilian Stock Index)      | Principal índice de desempenho das ações negociadas na B3.                     |
| Certificado de Depósito Interbancário | CDI            | Interbank Deposit Certificate         | Taxa de juros de referência para operações interbancárias. Benchmark comum.    |
| Documento de Arrecadação de Receitas Federais | DARF   | Federal Revenue Collection Document   | Guia para pagamento de impostos federais (e.g., IR sobre ganhos de capital). |
| Come-cotas                        | -                | Quota-eater (Fund Taxation)           | Antecipação semestral do IR em alguns fundos de investimento.                 |
| Análise do Perfil do Investidor   | API              | Investor Profile Analysis (Suitability) | Procedimento obrigatório (CVM) para adequação de produtos ao investidor.      |
| Realização de Prejuízo Fiscal     | -                | Tax-Loss Harvesting                   | Estratégia de vender ativos com prejuízo para compensar ganhos tributáveis.    |
| Ações                             | -                | Stocks / Equities                     |                                                                              |
| Renda Fixa                        | -                | Fixed Income                          |                                                                              |
| Renda Variável                    | -                | Variable Income / Equities            |                                                                              |
| Previdência Privada               | -                | Private Pension Plan                  | Planos como PGBL e VGBL.                                                      |
| Dividendos                        | -                | Dividends                             |                                                                              |
| Juros Sobre Capital Próprio       | JCP              | Interest on Equity                    | Forma de distribuição de lucros de empresas, tributada na fonte.             |
| Alocação de Ativos                | -                | Asset Allocation                      |                                                                              |
| Carteira de Investimentos         | -                | Investment Portfolio                  |                                                                              |
| Liquidez                          | -                | Liquidity                             |                                                                              |
| Rentabilidade                     | -                | Profitability / Return                |                                                                              |
| Volatilidade                      | -                | Volatility                            |                                                                              |
| Debêntures                        | -                | Debentures / Corporate Bonds          | Títulos de dívida emitidos por empresas (não financeiras).                   |
| COE (Certificado de Operações Estruturadas) | COE    | Structured Notes                      | Instrumento financeiro que combina renda fixa e variável.                    |
| ETF (Exchange Traded Fund)        | ETF              | Exchange Traded Fund                  | Fundo de investimento negociado em bolsa como uma ação.                      |
| CVM (Comissão de Valores Mobiliários) | CVM            | Brazilian Securities and Exchange Commission | Autarquia que regula o mercado de capitais no Brasil.                        |
| BACEN (Banco Central do Brasil)   | BCB / BACEN      | Central Bank of Brazil                | Autoridade monetária do Brasil.                                              |
| LGPD (Lei Geral de Proteção de Dados Pessoais) | LGPD   | Brazilian General Data Protection Law | Lei de proteção de dados do Brasil.                                          |
| Suitability                       | -                | Suitability                           | Adequação do produto ao perfil do investidor.                                |
| Resiliência Comportamental (RISA) | RISA             | Behavioral Resilience (RISA)          | Framework para avaliação do perfil do investidor.                            |
| Modelo 3-L (Liquidez, Longevidade, Legado) | 3-L Model | 3-L Model (Liquidity, Longevity, Legacy) | Modelo de planejamento financeiro por objetivos.                               |

Este glossário será expandido conforme necessário.
